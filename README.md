# amazing.github.io

E-Commerce project using Nodejs, Express, EJS and Mongoose.

- MVC
- RESTful
- CRUD


User features:

- View Products Without Login
- Log-In 
- View Products
- Add Product to Cart
- Customize Product
- Create Product
- Delete Product
